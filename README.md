# EC Site Sample

## Get started

Run the following command to start the server.

```bash
make run
```

## Development

### Install dependencies

```bash
make install
```

### Run tests

```bash
make test
```

### Run linter

```bash
make lint
```

### Run formatter

```bash
make fmt
```

### Run server

```bash
make run
```

### Run server with hot reload

```bash
make dev
```

### Run server with debug mode

```bash
make debug
```

### Generate code automatically

```bash
make generate
```
